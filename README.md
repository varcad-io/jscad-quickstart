# JSCAD Quickstart

[Launch this project in the Varcad IDE](https://varcad.io/varcad-io/jscad-quickstart)

A minimal introduction to building parametric geometry with JSCAD in Varcad.

## What this demonstrates

- importing `@jscad/modeling`
- creating and positioning a primitive
- returning renderable geometry from `main()`
- reading interactive widget values

Open the project in Varcad, adjust the width, depth, and height controls, and render the result.

Upstream documentation: [JSCAD Documentation](https://openjscad.xyz/docs/)
